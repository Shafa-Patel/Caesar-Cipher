# Caesar-Cipher
A simple Python implementation of the classic Caesar Cipher with CLI examples and artistic output.
Project: Caesar Cipher (Python)
Summary: Lightweight Python implementation of the Caesar cipher for encrypting and decrypting text. Includes a CLI example (main.py) and optional ASCII art (art.py) for demonstration. Ideal for learning basic cryptography, string manipulation, and command-line scripting.
Features: Simple encrypt/decrypt functions, configurable shift/key, example runner, minimal dependencies.
